# SCT_CS_3 - Password Strength Checker

## Objective

To evaluate the strength of a password based on predefined security criteria.

## Description

This project checks whether a password contains uppercase letters, lowercase letters, numbers, special characters, and meets minimum length requirements. Based on the analysis, the password is classified as Weak, Medium, or Strong.

## Technologies Used

* Python
* Regular Expressions (re module)

## Input

Password: ujeA@125

## Output

Strong Password

## Result

The entered password met all required security conditions and was classified as a Strong Password.
